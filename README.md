**Toda la información de todas las películas del mundo en un sólo lugar.**
## CARACTERÍSTICAS PRINCIPALES:
- Guía interactiva
- Pantalla de búsqueda
- Pantalla de perfil
- Pantalla de recomendador
- Pantalla de información de elemento "Película"
- Pantalla de información de elemento "Persona"

## TECNOLOGÍAS EMPLEADAS:
- SharedPreferences
- Firebase
- Androidx Credentials 
- API The Movie DataBase (TMDB)

## INSTRUCCIONES DE USO:
IMPORTANTE: Si descarga una versión externa de la app diferente a la de Google Play, el inicio de sesión con Google no funcionará.
